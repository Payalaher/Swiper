Swiper

Swiper is the most modern free and open source mobile touch slider with hardware accelerated
transitions and amazing native behavior. Use it on websites, web apps, and mobile native/hybrid
apps.

Swiper, along with other great components, is a part of Framework7 and Ionic Framework7 - a fully-
featured frameworks for building iOS & Android apps.

Powered With Top Notch Features

Library Agnostic

Swiper doesn't require any JavaScript libraries like jQuery, which makes Swiper much smaller and faster

Modular Structure

Swiper has a modular structure and allows you to use only the necessary modules to make its size even smaller

Rich API

Swiper comes with a very rich API. You can even create your own plugins

Full RTL Support

Swiper provides 100% RTL support with correct layout

Multi Row Slides Layout

Swiper allows a multiple row slides layout, with a few slides per column

Transition Effects

There are a lot of transition effects including 3D effects

Two-way Control & Thumbs

Swiper can be used as controller for any number of other Swipers

Full Navigation Control

All required navigation elements are here - Pagination, Navigation, Scrollbar

Most Flexible Slides Layout Grid

Swiper has a lot of parameters to make it as flexible as possible

Parallax Transitions

Swiper supports modern parallax transition effects that can be used on any element inside of Swiper: images, text blocks, headings, backgrounds, etc.

Images Lazy Loading

Delays loading of images in inactive/invisible slides until the user swipes to them

Virtual Slides

This feature is great when you have a lot of content-heavy slides so it keeps just the required amount of slides in DOM

CSS Scroll Snapping

Special mode that uses natural CSS scrolling for unlimited performance

And many more ...

Responsive, scroll prevention, resistance, autoplay, loop mode, etc.
